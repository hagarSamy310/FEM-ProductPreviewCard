### Overview

A fully responsive Product Preview Card Component, a Frontend Mentor challenge.
The goal was to recreate the provided design as closely as possible using semantic HTML and modern CSS practices.

The card showcases a product image, description, and pricing details with a clean and minimalist layout.
It adapts seamlessly across different screen sizes, stacking vertically on mobile and displaying a side-by-side layout on larger screens.
